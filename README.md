# Exercicios de Programação Web 
Exercícios desenvolvidos na disciplina de Programação Web do curso de Sistemas de Informação no CEFET-RJ.

## 📚 Conteúdo abordado
- APIs REST
- Rotas e Middlewares
- Integração com banco de dados
- Arquitetura em camadas
- Desenvolvimento de interfaces web

## 🛜 Tecnologias 
### 🏦 Backend
- Node.js
- Express
- Sequelize
- Knex.js
- NestJS

### 💻 Frontend
- React
- Next.js
- Tailwind CSS
- shadcn/ui
